# Описание проекта
Интернет-магазин на Vue.js с использованием REST API.

Поведение.
Пользователь  может выполнить на сайте следующие действия:
* Посмотреть список товаров по страницам.
* Отфильтровать список товаров по параметрам.
* Посмотреть информацию о товаре на его детальной странице.
* Добавить товар с выбранным цветом и размером в корзину.
* Изменить количество товаров в корзине.
* Удалить товары из корзины.
* Внести данные на странице оформления заказа.
* Выбрать способ доставки и оплаты.
* Оформить заказ и посмотреть информацию о нём на странице успешного оформления заказа.

Состав продукта
* Список товаров.
* Страница товара.
* Корзина.
* Оформление заказа.
* Успешное оформление заказа.

# Развертывание приложения на вашем ПК

1. Клонируем репозиторий к себе на компьютер и переходим в соответствующую папку.
2. Устанваливаем прокет введя в терминал команду npm install.
3. Запускам приложение введя в терминал команду npm run serve. Обычно приложение становится доступным по адресу http://127.0.0.1:8080/ 

Чтобы собрать продакшн версию приложения запустите через терминал команду npm run build.
Собранное приложение будет располагаться в репозитории в папке dist.
